## What is this?

**AI Visual Prompt Gallery** is a curated prompt reference and visual inspiration site for creators who want to study how strong AI visuals are structured.

Instead of only collecting text prompts, the gallery is organized around the final visual result: composition, style, subject, lighting, material, aspect ratio, source, model information and prompt structure.

It is useful when you want to quickly answer questions like:

- What kind of wording creates a stronger poster layout?
- How do product prompts describe material, lighting and camera angle?
- What does a good character sheet prompt usually include?
- How can I search by idea instead of exact keywords?
- How do image and video references differ in prompt structure?

## Why it exists

Prompt references are easier to learn when you can see the result first.

Many prompt libraries are text-heavy. This project is built around a more practical workflow:

1. Browse visual references.
2. Open a detail page that keeps the prompt, source, model, ratio and collection metadata together.
3. Study how the prompt describes subject, scene, style, composition, material, lighting and constraints.
4. Search by natural language when keywords are not enough.

The goal is not to replace creative thinking. The goal is to make visual prompt structure easier to study.

## Prompt Study Notes

A strong visual prompt usually does more than name a subject. It often contains several layers:

```text
Subject -> scene -> composition -> style -> material -> lighting -> camera -> constraints
```

Example patterns you will see in the gallery:

```text
Create a vertical 2:3 travel poster with an aged-paper Art Deco style...
```

```text
Create a premium collectible object with enamel texture, soft studio lighting and packaging detail...
```

```text
Create a photorealistic wide-angle interior visualization with annotation callouts...
```

These patterns are not formulas. They are starting points for learning how visual intent is organized.
