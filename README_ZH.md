这是 `soulg.xyz` 的公开项目说明页。

本站是一个面向 AI 视觉创作者的灵感图库，整理图片、视频、海报、角色、产品视觉和 PPT 方向的公开创作案例。它的重点不是单纯堆提示词，而是把最终画面、提示词、模型、来源、比例和采集信息放在一起，方便创作者学习画面表达方式。

## 27,000+ 视觉灵感资源，持续更新中

图库目前整理了两万七千多个图片、视频和 PPT 视觉参考，并会继续补充新的公开灵感案例。重点不是“收藏得多”，而是让创作者能快速看到画面结果，再回到详情页研究提示词结构。

## 它适合做什么

- 找图片、视频和海报方向的创作灵感
- 学习提示词如何描述主体、场景、构图、光影、材质和限制条件
- 用语义搜索找到接近创作意图的画面
- 对比不同模型和不同来源的视觉表达方式
- 做 prompt 写作练习和创作参考整理
